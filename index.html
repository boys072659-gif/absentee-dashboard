💡 사용 가이드
1️⃣ 봇을 텔레그램 그룹방에 초대합니다.
2️⃣ 방에서 /start 실행 → 봇이 Chat ID를 안내합니다.
3️⃣ 해당 Chat ID를 여기에 입력하여 승인합니다.
4️⃣ 방에서 /start 재실행 → 이제 사용 가능합니다.

⚠️ 승인 해제(비활성) 하면 해당 방은 즉시 사용 차단됩니다.
